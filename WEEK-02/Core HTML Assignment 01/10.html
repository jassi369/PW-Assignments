1. HTML

    Primary use cases Key Features/Benefits
    Creating and structuring content for websites and webpages.
    • Simplified markup language for web content.
    • Supports multimedia integration and interactivity.
    • Platform-independent and widely supported by browsers.

  2. CSS

    Primary use cases Key Features/Benefits
    Styling and designing the appearance of web pages.
    • Separates content from presentation.
    • Provides responsive and adaptive layouts.
    • Supports consistent design across multiple pages.

  3. JavaScript

    Primary use cases Key Features/Benefits
    Adding interactivity and dynamic behavior to web pages.
    • Enables client-side web page manipulation.
    • Versatile for frontend and backend development.
    • Supports extensive libraries and frameworks.
