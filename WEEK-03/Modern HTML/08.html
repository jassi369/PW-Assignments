What is html?

HTML, or HyperText Markup Language, is a markup language that is used to create web pages. It is a text-based language that uses tags to define the structure of a web page.

What is semantic html?

Semantic HTML is a way of writing HTML that makes it easier for web browsers and other software to understand the meaning of the content on a web page. This can help to improve the accessibility of web pages, as well as the way that they are displayed in search engine results pages.

Common Semantic HTML tags?

  • <header>: Define a header section.
  • <nav>: Define navigation links.
  • <main>: Define main thnigs of a document.
  • <section>: Define different section of document.
  • <article>: Define article that can be shared independently.
  • <aside>: Define content that is aside from the content
  • <footer>: Define a footer for the page.
